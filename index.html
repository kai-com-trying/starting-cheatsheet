CSS Colours

CSS Color Names

CSS provides a collection of predefined color names, such as red, blue, green, and black, that can be used to apply colors to elements. These color names are human-readable and widely supported across all browsers. While limited in variety compared to other color formats, they are a quick and simple way to set basic colors.

Check out MDN Web Docs for a list of CSS named colors.

Hex Codes

Hexadecimal color codes are a way of representing colors using a six-character code, combining red, green, and blue (RGB) values. Each pair of characters represents one of the primary colors (red, green, or blue) using values ranging from 00 to FF (hexadecimal), where 00 is the minimum intensity and FF is the maximum intensity.

RGB Codes

RGB color codes represent colors as a combination of red, green, and blue light intensities. Each color component is represented by a number between 0 and 255, where 0 is the absence of that color and 255 is full intensity. This format also supports an extended version, rgba(), where the a stands for alpha (opacity), allowing for transparent colors.

HSL Codes

HSL represents colors based on their hue, saturation, and lightness.

  • Hue is the color itself, represented as an angle on the color wheel (0–360), where 0 is red, 120 is green, and 240 is blue.
  • Saturation controls the intensity or purity of the color, with 0% being a shade of gray and 100% being fully saturated.
  • Lightness determines how light or dark a color is, with 0% being black and 100% being white. Similar to RGB, HSL also has an extended format called hsla(), where the a stands for alpha (opacity).
